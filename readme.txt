安装环境要求：

操作系统：Windows 10/11
后端：Node.js 14 及以上版本
数据库：MySQL
前端：Hbulider，支持现代浏览器（Chrome、Firefox、Edge）

安装过程：

后端环境搭建：
安装 Node.js（https://nodejs.org/）
进入项目目录：cd project_folder
安装依赖：npm install

数据库配置：
安装并启动 MySQL
在配置文件
(项目路径\springboot\src\main\java\com\example\SpringbootApplication.java)
中设置连接数据库的用户名、密码等信息
使用 source 你的路径\history.sql 运行 sql 脚本
(运行前确保 mysql 中没有其他来源的名为 history 的库，脚本会占用该库名)
如使用 source 指令运行时提示错误，可使用 use history;然后将 sql 脚本中 insert 插入语句按照从上到下的顺序单独复制执行。

启动后端服务：
使用命令行跳转到项目路径\springboot（后端项目目录）
执行 mvn clean package
使用 cd target 跳转到 target 文件夹
执行 java -jar springboot-0.0.1-SNAPSHOT.jar

前端部署：
使用 HBuilder 打开前端目录(项目路径\daynasty)
运行至浏览器

典型使用流程：
进入首页，浏览各个朝代简介、信息，
点击朝代图片进入朝代页面，显示该朝代的政权和君王列表。
在朝代界面中查看君王信息与各种历史事件。
点击感兴趣的历史事件，浏览事件的详细经过。
收藏感兴趣的内容，留下自己的评论。
